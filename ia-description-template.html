Overview

The data is stored in NDJSON (Newline Delimited JSON) format, compressed with XZ. Each line represents a single post object as received from the desuarchive API [1].

Notes [1]

  • Data from archive.heinessen.com was converted to roughly look like the desuarchive API response.
  • Same goes for data from 4archive.org.
  • arch.b4k.dev and archived.moe run the same software as desuarchive (FoolFuuka) so their responses were already in "desuarchive format".
  • Posts that could not be retrieved from any known source are represented by a placeholder JSON object to maintain continuity (schema described below).

Missing Post Schema
